# download-remote-file-to-server
PHP를 통해 원격지 파일 다운로드를 합니다.

## 사용방법
URL에 http://example.com/test.zip 같은 형식의 URL을 넣고 다운로드를 해주세요.<br>
다운로드 완료시 files폴더에 저장합니다.<br>
복사버튼은 접속중인 도메인과 URL을 이용하여 자동으로 표시합니다.

## 설치방법
딱히 없습니다.
